20201109 Hugo で github pages にコンテンツを公開する

November 09, 2020    

資料

  • Hugo + GitHub Pages + 無料で洒落たブログを30分で作る - Qiita
  • Quick Start | Hugo
  • Host on GitHub | Hugo

手順

Hugo のインストール

brew install hugo

サイト作成

hugo new site $USER.github.io
cd $USER.github.io

テーマ設定

今回は Simpleness | Hugo Themes を使う

echo 'theme = "pickles"' >> config.toml
git submodule add https://github.com/mismith0227/hugo_theme_pickles themes/pickles

プロジェクト設定

必要に応じて書き換える

publishDir は github pages の対象ディレクトリなので docs 固定

baseURL = "https://i524.github.io/"
languageCode = "ja-jp"
title = "1日1時間"
theme = "simpleness"
publishDir = "docs"

コンテンツ追加

hugo new blog/20201109-hugo-github-pages.md

動作確認は以下で

hugo server -D
hugo
